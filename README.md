# 键盘红绿灯 - 久坐提醒工具

## 概述

基于键盘活动的红绿灯久坐提醒工具。通过监听键盘输入，用红绿灯和行人动画可视化工作状态，提醒用户适时休息。

## 核心功能

### 1. 三阶段状态机

```
绿灯（工作中）→ 黄灯（该休息了）→ 红灯（必须休息）
    ↑                                    │
    └────────── 休息确认 ←───────────────┘
```

- **绿灯阶段**：50分钟，显示行人走路动画
- **黄灯阶段**：10分钟，提示需要休息
- **红灯阶段**：直到用户确认休息，强制提醒

### 2. 行人走路动画

- 根据键盘敲击速度控制行人走路快慢
- 打字快 → 行人走快（小跑）
- 打字慢 → 行人走慢
- 不打字 → 行人静止

### 3. 休息确认机制

- 快捷键确认：`Ctrl+Alt+R` 重置计时器
- 自动检测：3分钟无键盘活动自动判定休息

## 技术架构

### 文件结构

```
keyboard-traffic-light/
├── DESIGN.md              # 本文档
├── config.json            # 配置文件
├── traffic_light.ps1      # 主程序（UI + 逻辑）
├── patterns.ps1           # 灯语引擎
├── keyboard.ps1           # 键盘监听模块
├── timer.ps1              # 定时器逻辑
├── 启动.bat               # 启动脚本
```

### 数据流

```
键盘输入
   ↓
GetAsyncKeyState (16ms轮询)
   ↓
打字速度计算（滑动窗口）
   ↓
行人动画速度
   ↓
定时器状态机（50min→10min→休息）
   ↓
红绿灯状态更新
```

### 配置项

```json
{
  "workDuration": 50,        // 工作时长（分钟）
  "warningDuration": 10,     // 黄灯警告时长（分钟）
  "restThreshold": 180,      // 休息判定阈值（秒）
  "typingSpeedWindow": 5,    // 打字速度计算窗口（秒）
  "window": { "x": 100, "y": 100 }
}
```

## 状态定义

| 状态 | 灯光 | 行人动画 | 含义 |
|------|------|----------|------|
| working | 绿灯常亮 | 根据打字速度走路 | 正常工作中 |
| warning | 黄灯闪烁 | 静止 | 该休息了 |
| rest | 红灯闪烁 | 静止 | 必须休息 |

## 打字速度映射

| 按键频率 | 行人状态 | 动画速度 |
|----------|----------|----------|
| 0 次/秒 | 静止 | 0x |
| 1-2 次/秒 | 慢走 | 0.5x |
| 3-5 次/秒 | 正常走 | 1x |
| 6+ 次/秒 | 快走/小跑 | 2x |
