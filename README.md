# 键盘红绿灯 - 久坐提醒工具

## 概述

基于键盘活动的红绿灯久坐提醒工具。通过监听键盘输入，在红绿灯中用火柴人动画可视化工作状态，提醒用户适时休息。

## 核心功能

### 1. 三阶段状态机

```
绿灯（工作中）→ 黄灯（该休息了）→ 红灯（必须休息）
    ↑                                    │
    └────────── 休息确认 ←───────────────┘
```

- **绿灯阶段**：50分钟，火柴人坐着敲键盘，前方有电脑屏幕
- **黄灯阶段**：10分钟，黄灯闪烁，火柴人循环伸懒腰提醒休息
- **红灯阶段**：红灯闪烁，火柴人循环站起来扔掉电脑，强制休息

### 2. 三个火柴人动画

| 灯色 | 火柴人状态 | 动画效果 |
|------|-----------|----------|
| 绿灯 | 坐着敲键盘 | 打字快→手臂动得快，打字慢→手臂动得慢，不打字→静止 |
| 黄灯 | 坐着伸懒腰 | 黄灯亮时循环播放：手臂上举→身体后仰→恢复（2.5秒循环） |
| 红灯 | 站起来扔电脑 | 红灯亮时循环播放：站起→抓住屏幕→斜向扔出→电脑消失→瞬间刷新（2秒循环） |

### 3. 电脑元素

每个灯内的火柴人前都有一台小电脑（屏幕 + 支架 + 键盘），以火柴人身体 X=20 为对称轴居中放置。

```
  ┌────┐
  │屏幕│  ← 显示器 10x7，上沿对齐手下沿 Y=22
  └────┘
    │     ← 支架 X=20，从屏幕底部到座椅线
  ══════  ← 座椅线 Y=30 + 键盘
```

### 4. 休息确认机制

- 快捷键确认：`Ctrl+Alt+R` 重置计时器
- 自动检测：3分钟无键盘活动自动判定休息

## 技术架构

### 文件结构

```
keyboard-traffic-light/
├── DESIGN.md              # 本文档
├── config.json            # 配置文件
├── traffic_light.ps1      # 主程序（UI + 动画 + 状态机 + 定时器）
├── patterns.ps1           # 灯语引擎（红灯闪烁/绿灯呼吸等模式）
├── keyboard.ps1           # 键盘监听模块（GetAsyncKeyState）
├── 启动.bat               # 启动脚本
```

### 数据流

```
键盘输入
   ↓
GetAsyncKeyState (16ms轮询)
   ↓
打字速度计算（滑动窗口）
   ↓
打字动画速度
   ↓
定时器状态机（50min→10min→强制休息）
   ↓
红绿灯状态 + 火柴人动画更新
```

### 配置项

```json
{
  "workDuration": 50,           // 工作时长（分钟）
  "warningDuration": 10,        // 黄灯警告时长（分钟）
  "restDuration": 5,            // 红灯休息倒计时（分钟）
  "restThreshold": 180,         // 旧：休息判定阈值（秒）
  "typingSpeedWindow": 5,       // 打字速度计算窗口（秒）
  "window": { "x": 100, "y": 100 }
}
```

## 状态定义

| 状态 | 灯光效果 | 火柴人动画 | 下方文字 |
|------|---------|-----------|---------|
| working | 绿灯常亮 | 坐着敲键盘，速度随打字频率 | "Working MM:SS" (白色) |
| warning | 黄灯闪烁 | 坐着循环伸懒腰 | "Break MM:SS" (黄色) |
| rest | 红灯闪烁→常亮 | 循环扔电脑→静止等待 | "Rest" + 倒计时 (红色) |

## 打字速度映射

| 按键频率 | 手臂敲键盘状态 | 动画速度 |
|----------|--------------|----------|
| 0 次/秒 | 静止 | 0x |
| 1-2 次/秒 | 慢速敲击 | 0.5x |
| 3-5 次/秒 | 正常敲击 | 1x |
| 6+ 次/秒 | 快速敲击 | 2x |

## 灯语模式

| 模式ID | 名称 | 效果 |
|--------|------|------|
| 0 | 绿灯常亮 | Green=1.0 |
| 2 | 黄灯慢闪 | Green 方波 1000ms |
| 3 | 黄灯快闪 | Green 方波 500ms |
| 4 | 红灯闪烁 | Red 方波 800ms |
| 5 | 红灯火心跳 | Red 心跳脉冲 |

## 动画时序

### 黄灯伸懒腰动画（2.5秒循环）

| 阶段 | 时间 | 动作 |
|------|------|------|
| 展开 | 0~1.0s | 双臂上举展开，身体后仰，头部微抬 |
| 放松 | 1.0~2.5s | 保持伸展姿态，轻微放松 |

### 红灯扔电脑动画（2.0秒循环）

| 阶段 | 时间 | 动作 |
|------|------|------|
| 站起 | 0~0.6s | 站起来，右手伸向电脑屏幕 |
| 抓住 | 0.6~1.1s | 单手抓住屏幕，向上拽起 |
| 扔出 | 1.1~1.6s | 手臂斜向右上方挥出，屏幕飞出去 |
| 刷新 | 1.6~2.0s | 保持扔姿势，0.1s后瞬间刷新回坐姿+新电脑 |

### 红灯休息倒计时

| 阶段 | 说明 |
|------|------|
| 倒计时中 | 红灯闪烁 + 火柴人循环扔电脑 + "Rest" + 倒计时 |
| 倒计时结束 | 红灯常亮 + 火柴人静止坐姿 + "Rest / done" |
| 敲击任意键 | 切换到绿灯，重新开始工作周期 |
